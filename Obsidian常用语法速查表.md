# Obsidian 常用语法速查表

适用范围：Obsidian 默认支持的常用 Markdown 语法，以及 Obsidian 自带的双链、嵌入、Callout 等高频功能。

---

## 1. 标题

```md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

效果：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

---

## 2. 文本样式

| 功能 | 语法 | 示例效果 |
|---|---|---|
| 粗体 | `**粗体**` | **粗体** |
| 斜体 | `*斜体*` | *斜体* |
| 粗斜体 | `***粗斜体***` | ***粗斜体*** |
| 删除线 | `~~删除线~~` | ~~删除线~~ |
| 高亮 | `==高亮==` | ==高亮== |
| 行内代码 | `` `code` `` | `code` |

## 3. 段落与换行

```md
这是第一段。

这是第二段。
```

说明：

- 空一行表示新段落。
- 如果需要强制换行，可以在行尾加两个空格，或直接另起一行。

## 8. 链接

外部链接：

```md
[OpenAI](https://openai.com)
```

内部笔记链接：

```md
[[笔记名]]
```

自定义显示文字：

```md
[[笔记名|显示名称]]
```

链接到标题：

```md
[[笔记名#某个标题]]
```

链接到当前笔记标题：

```md
[[#某个标题]]
```

---

## 9. 块引用链接

先给某一段添加块 ID：

```md
这是一个重要段落。 ^block-id
```

再链接到它：

```md
[[笔记名#^block-id]]
```

---

## 10. 嵌入内容

嵌入整篇笔记：

```md
![[笔记名]]
```

嵌入标题：

```md
![[笔记名#某个标题]]
```

嵌入段落块：

```md
![[笔记名#^block-id]]
```

嵌入图片或附件：

```md
![[image.png]]
![[image.png|300]]
```

说明：

- `!` 表示“嵌入显示”，不是普通跳转链接。
- `|300` 常用于设置图片显示宽度。

## 15. Callout 提示框

```md
> [!note] 笔记
> 这里是普通说明

> [!tip] 提示
> 这里是使用技巧

> [!warning] 警告
> 这里是注意事项
```

常见类型：

- `note`
- `abstract`
- `info`
- `tip`
- `success`
- `question`
- `warning`
- `failure`
- `danger`
- `bug`
- `example`
- `quote`

可折叠写法：

```md
> [!tip]- 点我折叠
> 折叠内容写这里
```

## 19. YAML 属性区

常放在笔记最顶部：

```md
---
title: Obsidian 常用语法速查表
tags:
  - Obsidian
  - Markdown
aliases:
  - Obsidian 语法
---
```

常见用途：

- 设置标题、别名、标签
- 给 Dataview / Bases / 查询类插件提供结构化元数据
- 统一管理笔记属性

## 20. 实用组合示例

### 示例 1：会议记录模板

```md
---
tags:
  - 会议
  - 项目A
---

# 2026-04-18 项目例会

## 议题
- [ ] 进度同步
- [ ] 风险确认

## 结论
> [!summary]
> 本次会议确定下周交付初版。

## 待办
- [ ] 张三：整理文档
- [ ] 李四：完成测试
```

### 示例 2：知识笔记模板

```md
---
tags:
  - 学习
  - Obsidian
---

# 双向链接

定义：用于连接笔记之间的关系。

相关笔记：
- [[知识管理]]
- [[Markdown 基础]]

重点结论：这是核心概念。 ^key-point

引用这个重点：
![[#^key-point]]
```

## 22. 使用建议

- 日常记笔记优先掌握：标题、列表、任务、双链、嵌入、标签。
- 做知识管理优先掌握：`[[双链]]`、块引用、YAML 属性、Callout。
- 做项目管理优先掌握：任务列表、标签、模板、附件嵌入。
- 如果你主要在 Obsidian 中写作，建议把这份文件固定到侧边栏或收藏。
